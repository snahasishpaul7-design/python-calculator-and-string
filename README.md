## 🌟 Project Overview

This is an **advanced Python-based chatbot system** designed to simulate an interactive AI-like assistant inside the terminal.

It can understand user input, perform calculations, open websites, play games, and provide information in real time.

---

## ✨ Key Features

### 💬 Smart Chat System
- Natural conversation handling
- Fuzzy input recognition
- Random intelligent responses

### ➗ Math Engine
- Basic arithmetic operations
- Advanced math (sin, cos, tan, log, factorial)
- Power, square root, ceil, floor

### 🎮 Game Module
- Number guessing game
- Random range generation
- Unlimited attempts

### 🌐 Web Integration
- Open Google, YouTube, ChatGPT
- Direct search functionality

### 🌍 Country Information System
- Capital city
- Languages
- Borders
- Currency
- Timezone

### 🔐 Utilities
- Password generator
- Fake data generator
- Joke system

---

## 🛠️ Tech Stack

| Library | Purpose |
|--------|--------|
| `random` | Random number generation |
| `math` | Mathematical operations |
| `re` | Pattern matching |
| `pyjokes` | Joke system |
| `pyfiglet` | ASCII banner text |
| `colorama` | Terminal colors |
| `faker` | Fake data generation |
| `countryinfo` | Country details |
| `fuzzywuzzy` | Smart input matching |
| `webbrowser` | Open websites |
